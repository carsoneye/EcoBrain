# Analysis Files Rules

- Every finding must reference file paths (and line numbers where relevant)
- Observations only — no opinions without evidence
- Link findings to the project's `decisions/recommendations.md` where applicable
- Bullets and tables over prose. Optimize for scanning.
- Resolved items get `[resolved YYYY-MM-DD]`, never deleted
