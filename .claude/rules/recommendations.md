# Recommendations Rules

- Every recommendation traces back to a specific finding in analysis/
- Include effort estimate (S/M/L) and impact rating (Low/Med/High)
- Sort by impact desc, then effort asc (quick wins first)
- When a recommendation is implemented, mark it `[x]` and note the date
- Don't delete completed recommendations — they're a record of improvements
